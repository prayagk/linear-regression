## Getting Started

1. Clone the repo
   ```bash
   git clone https://github.com/prayagk/linear-regression.git
   cd linear-regression
   ```
2. Install dependencies
   ```bash
    npm install
   ```
3. Start the development server
   ```bash
    npm run dev
   ```
